HTML-HyperText Markup Languages

Elments-building blocks of HTML pages

  • Paired tagging:  <h1>inner texts</h1>
  • Unpaired tagging:  <img><br><hr>

Attribute-written inside of the angle bracket (start tag)

  1. id: unique identifier for an element
  2. class: designate a name to style elements
  3. style: style elements inline

Cmmonly used elements:

<div><h1><p><img><button><input><select><a><li>
