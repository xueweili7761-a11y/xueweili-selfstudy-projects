IDE: integrated development environment (VSC, webstorm)
Settings: appearance (dark mode), plugin (VIM, Rainbow...), terminal

VIM:
i ====> insert mode, you can type
esc ===> control mode
Under control mode:
gg -> locate 1st line of code
G -> locate last line of code
dd -> delete 1 line
5dd -> delete 5 lines
p -> paste
o -> insert a new line below
O -> insert a new line above
yy -> copy 1 line
: + line number -> locate to the designated line
A -> move cursor to end of the line
I -> move cursor to start of the line

pwd:print your working directory
